## ⚡ Run in 3 Steps

### 1️⃣ Open PowerShell in this folder
Right-click this folder → "Open in Terminal" or "Open PowerShell window here"

### 2️⃣ Install dependencies (first time only)
```powershell
npm install
```
⏱️ Takes about 10-20 seconds

### 3️⃣ Start the development server
```powershell
.\run-dev.ps1
```

**That's it!** 🎉 Open your browser to **http://localhost:3000**

---

## 🛑 Stop the Server
Press `Ctrl + C` in the terminal

---

## ❓ Need Help?

### Common Issues:

**Q: "npm install" fails?**  
A: Make sure Node.js is installed. Check with: `node -v`

**Q: Server won't start?**  
A: Use the helper script: `.\run-dev.ps1`

**Q: How to change colors?**  
A: Edit `index.html` → Find the Tailwind config section

**Q: How to add more projects?**  
A: Edit `constants.tsx` → Add to `projectsData` array

**Q: Can I use this for free forever?**  
A: **YES!** No API costs, no subscriptions, completely free! 🎉
